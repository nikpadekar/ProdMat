Angular Material inside an Angular module

You can find here an example,
how to use the Angular Material,
in a independant Angular module
Create an Angular module

I called angular-material.module for example.

 Packages and dependencies to install
  • @angular/material
  • @angular/cdk
  • @angular/animations
    This one is optionnal, needed only for components,
    that use Angular Animations
    In this case it's mandatory to add,
    BrowserAnimationsModule in angular-material.module
Import angular-material components

Inside Imports array, import the component you want,
then import this module inside the application module.

 Include a theme

Which is required for some core functions, and can be added easily, by adding inside the main style file the line:

 @import "~@angular/material/prebuilt-themes/indigo-pink.css"; Install the package HammerJS

For gesture support, for some components like the Slide or the Tooltip, you can add it using a CDN also, details on the Link. Or you can install the package and it to the project dependecies, and import it then to the file main.ts

 import 'hammerjs'; Material Icons
